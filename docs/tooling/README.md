# safldk-docs

Documentation build tooling for [safl.dk](https://safl.dk). Installing this
package (`pipx install ./tooling` from `docs/`) provides two commands:

- `safldk-docs-serve` - live-rebuild dev server on http://localhost:8000
- `safldk-docs-build-html` - one-shot HTML build into `docs/_build/html/`

Run the commands from the `docs/` directory (the one containing `src/conf.py`).
